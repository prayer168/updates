# 黑熊老師網站優化檢核表｜表格式 GitHub 部署版

本版依照 Word 檢核表的正式版面重製，共 101 項。

## 功能
- 表格式呈現：優先、完成、進行中、未完成、不適用、檢核項目、驗收標準／備註
- 每個狀態按一下即可切換
- 備註自動儲存
- 檢核日期、人員、階段與驗收總結可遠端儲存
- 自動計算完成百分比
- 手機自動轉為卡片式排列

## 更新既有 Supabase
1. 在 Supabase SQL Editor 執行新的 `supabase.sql`
2. 將原本已填好的 `config.js` 複製到本資料夾，或重新填入：
   - Project URL
   - Publishable key
3. 本機測試後再上傳 GitHub Pages

## 注意
此版本仍是公開共用資料模式。知道網址的人都可以修改紀錄。
